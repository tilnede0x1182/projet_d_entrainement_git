# Projet d'entraînement Git – Gestion de conflits

Ce dépôt a été créé dans le but de simuler un projet collaboratif avec plusieurs développeurs, afin de m'entraîner aux situations concrètes de gestion de branches, de fusions (`merge`), de rebase, et surtout de résolution de conflits Git.

Le projet ne contient pas de code applicatif réel, mais sert de terrain d’expérimentation méthodique pour :
- Comprendre les conflits de versions lors de merges simultanés
- Manipuler les commandes Git dans des scénarios complexes
- Améliorer mes réflexes en environnement multi-collaborateurs

Chaque branche ou scénario correspondra à un cas pratique spécifique.

Je travaille seul sur ce dépôt, en simulant plusieurs utilisateurs avec des clones locaux et plusieurs terminaux.
